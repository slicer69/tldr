bastet

Clone of the game Tetris in the terminal.

More information: home page

- Start a tetris game:
bastet
- Navigate the piece horizontally:
Left|Right arrow key
- Rotate the piece clockwise or counterclockwise:
Spacebar|Up arrow key
- Soft drop the piece:
Down arrow key
- Hard drop the piece:
Enter
- Pause the game:
p
- Quit the game:
Ctrl + C
