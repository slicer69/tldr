drush

A command-line shell and scripting interface for Drupal.

More information: home page

- Download module "foo":
drush dl foo
- Download version 7.x-2.1-beta1 of module "foo":
drush dl foo-7.x-2.1-beta1
- Enable module "foo":
drush en foo
- Disable module "foo":
drush dis foo
- Clear all caches:
drush cc all
- Clear CSS and JavaScript caches:
drush cc css-js
