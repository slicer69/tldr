calibre-server


Description: A server application that can be used to distribute ebooks over a network.

Description: Ebooks must be imported into the library using the GUI or calibredb before.

Description: Part of the Calibre ebook library.

- Start a server to distribute ebooks. Access at http://localhost:8080:
calibre-server
- Start server on different port. Access at http://localhost:port:
calibre-server --port {{port}}
- Password protect the server:
calibre-server --username {{username}} --password {{password}}
